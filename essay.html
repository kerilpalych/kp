GitHub — крупнейший веб-сервис для хостинга IT-проектов и их совместной разработки. Основан на системе контроля версий Git и разработан на Ruby on Rails и Erlang компанией GitHub


Почему я выбрал GitHub: помимо его популярности, удобства в размещении и изменении кода, ГитХаб также предоставляет возможность делиться проектами, делать их совместно с отображением вклада каждого участника


Для анализирования посещаемости сайта я выбрал Яндекс Метрику, которая позволяет узнавать, сколько людей посещало сайт и другие показатели. Яндекс Метрику можно добавить на любой сайт, а код предоставляется самим Яндексом, что делает ее удобной и легкодоступной.
